Comandos linux 

ls - listar diretorio 
pwd - mostra caminho completo
clear - limpa o console
cd <nomeDaPasta> - entrar na pasta
cd .. - retornar para pasta anterior
mkdir <nomeDaPasta> - criar pasta

Comandos GIT 

git init - iniciar um repositório
rm -rf .git =: remover repostório

git config user.name -> validar usuágitrio
git config user.email -> validar email
git config user.name "nomeUsuario"
git add . -> adicionar todos os arquivos
git add nomeArquivo -> adicionar arquivo
git status - visualizar o status do repo
git commit -m "mensagem do commit"
git log -> visualizar a log
git remote add origin <url do git>
git branch -M <nome da branch>
git remote
